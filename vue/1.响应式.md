## 侵入式和非侵入式

Vue数据变化：

```js
this.count++
```

React数据变化

```'js
this.setState({
    count: this.state.count + 1
})
```

小程序数据变化:

```js
this.setData({
    count: this.data.count + 1
})
```



vue是非侵入式：

React和小程序是侵入式：setState不仅可以更新数据还可以更新视图。把视图的更新封装到了setState和setData里面
